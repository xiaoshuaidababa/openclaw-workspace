## 第六章：如何开始？

### 快速入门（10 分钟）
```bash
# 1. 安装
curl -fsSL https://openclaw.ai/install | bash

# 2. 初始化
openclaw onboard --install-daemon

# 3. 配置模型
# 选择 Anthropic/Google/OpenAI 等提供商
# 输入 API Key

# 4. 创建第一个工作流
openclaw cron add --name "我的第一个自动化" ...
```

### 设计原则
1. **幂等性**：同一触发器不产生重复副作用
2. **可观察性**：每次运行输出结构化日志
3. **容错机制**：失败时清晰告警，可恢复
4. **渐进式**：先自动化一个流程，稳定后再扩展
